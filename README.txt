#Steps to run the application
1. Import bookdetails project in eclipse and required actions need to be done(builing the project, etc).
2. Right click project -> Run as Java Application (or) 
mvn spring-boot:run -> run this command in command prompt.
3. Tomcat get started in the specified port.
4. Import nila-apps in visual studio(for front end) and needed action should be done(installing nodejs, etc).
5. Start the server using npm start in terminal.
6. It gets started in the port 4200.
7. Then run localhost:4200/books in browser.
8. Application shows the book details and the user can able to add new book details and sort the book details with respect to the choice given by the user.
